# BROCHURE GENERATOR

This is a brochure generator built using python. It uses llama3.2 model to generate brochure from company websites.

Built using:
- Python

To run it locally:
- clone this repo: `git clone git@github.com:NtwaliHeritier/brochure_generator.git`
- Run `uv sync` to compile the dependencies.
- Run `uv run main.py` to start to interact with the cli and follow the instructions.
